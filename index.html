Working Commands

  • !ping — Replies with Pong!
  • !hello — Greets you back.
  • !say <message> — Bot repeats your message.
  • !userinfo — Shows your Discord username and ID.
  • !help — Lists all available commands.
  • !time — Shows current server time.
  • !server — Displays server name and member count.
  • !bot — Displays the bot's name and status.
